# Secure Coding Basics

## Overview

This course covers C programming, memory handling, debugging, and basic cryptographic programming practices.

이 과목은 C 프로그래밍, 메모리 처리, 디버깅 및 기초 암호 프로그래밍을 학습합니다.

---

## Topics

- C programming
- Pointers and memory
- File processing
- SHA-256 hash practice
- Debugging
- Block cipher practice

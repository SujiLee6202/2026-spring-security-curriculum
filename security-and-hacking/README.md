# Security and Hacking

## Overview

This course covers the fundamentals of cybersecurity, including system security, network security, web security, cryptography, and incident response.

이 과목은 시스템 보안, 네트워크 보안, 웹 보안, 암호학 및 침해 대응 등 정보보안의 기초를 다룹니다.

---

## Topics

- System security
- Network security
- Web security
- Cryptography
- IDS / IPS
- Incident response

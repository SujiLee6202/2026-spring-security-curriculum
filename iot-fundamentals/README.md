# IoT Fundamentals

## Overview

This course introduces IoT technologies, embedded programming, communication protocols, and IoT security.

이 과목은 사물인터넷 기술, 임베디드 프로그래밍, 통신 프로토콜 및 IoT 보안을 다룹니다.

---

## Topics

- ESP32 programming
- Python programming
- GPIO / PWM
- UART / I2C / SPI
- MQTT and HTTP
- IoT security
